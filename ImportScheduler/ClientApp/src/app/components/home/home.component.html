ImportScheduler

Your CSV User import file shoud contain the following columns.
FirstName
Text required

The 'FirstName' property needs to be derived in the Import-CSV-File.

LastName
Text required

The 'LastName' property needs to be derived in the Import-CSV-File.

Gender
Text optional

You can specify the following values:
'm' for male
'f' for female
'n' for none

If no value is specified, the default value is 'n'

Email
Text required

Please provide a Email Address, that matches to the following pattern: user@domain.com

MobileNumber
Text optional

Please provide a Mobile Number, that matches to the following pattern: +1 717-827-7017
